Projects

Self Driving RC Car

Led a team of 4 for developing a self-driving RC car with real-time control using LPC 4078 RTOS board, utilizing motion planning algorithms, sensor data, and GPS signals on the CAN bus for motor control. The car has 4 nodes - sensor node and GPS node as input, motor node as the output and driver node for processing. Each node communicate with each other using CAN bus. The car avoids obstacles, and reach the provided destination using GPS. Shortest path is used for traversal from source to destination. GPS location is given through android app.

Technologies Used:

  • LPC 4078 Microcontroller used for test driven development.
  • RTOS and Periodic Scheduling were used to schedule tasks at different frequencies.
  • Motion Planning Algorithms for smooth movement of car with obstacle avoidance and GPS Navigation.
  • GPS, UART, I2C, BLuetooth, PWM, CAN Bus.
  • Coding is done in C.
  • Testing done using frameworks like CMock and Unity.
  • Sensors include ultrasonic sensor, Bluetooth module, Compass, GPS module, LCD, RPM sensor, motors.

Indoor Navigation for Shopping Robot

Individually developed a shopping robot for indoor environment with real-time control using LPC 1769. Several motion planning algorithms were developed for smooth navigation of any given indoor environment. Algorithms were developed for finding the territory of traversal, locating obstacles in the given area and converting it to a 2D map. Whenever destination location is given as input coordinates, robot moves to there, taking the shortest path and avoiding obstacles. Accelerometer and magnetometer was used develop the indoor coordinates and route map. This project received first prize in state government technical fest for robotics track.

Technologies Used:

  • LPC 1769 Microcontroller was interfaced with different sensors and coding was done in C.
  • Algorithms to find area, obstacles, mapping and shortest route traversal.
  • UART, I2C, PWM, Bluetooth, ADC

Enhanced Safety Device for Passengers

Developed a safety device along with 3 others to perform alcohol detection, sleep detection, call detection, person level identification and accident detection using PIC16F887 microcontroller and various sensors. The ignition is activated only if the driver is confirmed rid of alcohol. When the vehicle is running, continuous monitoring is carried out to detect accidents, incoming calls to the driver and whether the driver is sleeping . A mechanism to detect human presence inside a locked car is also implemented. Each method is used to rectify the carelessness of the driver and immediate intimation of accident is provided by the use of GSM technology. The system will send the accident location acquired from GPS along with the time. This will help the rescue service to reach in time and save the valuable human life. The simulation has been carried out in Proteus and the prototype is devised. .

Technologies Used:

  • PIC16F887 Microcontroller, GPS, GSM, GPIO.
  • Embedded C coding
